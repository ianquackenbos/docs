## Overview

Returns a summary of costs for a specific deployment, including total spend, month-over-month change, potential savings, and active recommendations.

## Request

<ParamField path="deployment_id" type="string" required>
  The unique identifier of the deployment
</ParamField>

### Example Request

```bash
curl -X GET "https://api.shardly.io/api/v1/costs/deployments/dep-123/overview" \
  -H "Authorization: Bearer $TOKEN"
```

## Response

<ResponseField name="total_monthly_spend" type="number">
  Total projected cost for the current month in USD
</ResponseField>

<ResponseField name="month_over_month_change" type="number">
  Percentage change from previous month (positive = increase)
</ResponseField>

<ResponseField name="potential_savings" type="number">
  Total potential savings from pending recommendations in USD
</ResponseField>

<ResponseField name="active_recommendations" type="integer">
  Number of pending recommendations
</ResponseField>

<ResponseField name="cost_breakdown" type="object">
  Cost breakdown by category
</ResponseField>

<ResponseField name="top_cost_drivers" type="array">
  Top 5 indices by cost
</ResponseField>

## Use Cases

### Dashboard Display

Display summary cards on the main dashboard:

```javascript
const { data: overview } = useCostOverview(deploymentId);

return (
  <div>
    <Card title="Monthly Spend">
      ${overview.total_monthly_spend.toLocaleString()}
    </Card>
    <Card title="vs Last Month">
      {overview.month_over_month_change > 0 ? '+' : ''}
      {overview.month_over_month_change}%
    </Card>
    <Card title="Potential Savings">
      ${overview.potential_savings.toLocaleString()}
    </Card>
  </div>
);
```

### Cost Alerting

Check for cost increases:

```python
overview = get_cost_overview(deployment_id)

if overview['month_over_month_change'] > 20:
    send_alert(
        f"Cost increase of {overview['month_over_month_change']}% detected"
    )
```

### Reporting

Generate monthly cost reports:

```python
for deployment in get_deployments():
    overview = get_cost_overview(deployment['id'])
    
    report.add_row({
        'deployment': deployment['name'],
        'cost': overview['total_monthly_spend'],
        'change': overview['month_over_month_change'],
        'savings_opportunity': overview['potential_savings']
    })
```
